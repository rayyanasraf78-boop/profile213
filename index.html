Tentang Saya

Saya adalah siswa jurusan Teknik Jaringan Komputer dan Telekomunikasi (TJKT) dari SMKS 1 Barunawati. Saya memiliki ketertarikan pada dunia jaringan komputer, perakitan perangkat keras, serta pengembangan website sederhana.

Projek yang Pernah Dilaksanakan

Merakit PC

Melakukan perakitan komputer dari komponen dasar hingga siap digunakan serta memastikan semua perangkat berfungsi dengan baik.

Instalasi Kabel Jaringan

Melakukan instalasi dan penataan kabel jaringan LAN, termasuk crimping kabel UTP dan pengujian koneksi.

Memahami Jaringan Komputer

Mempelajari dasar-dasar jaringan komputer seperti topologi jaringan, IP Address, dan perangkat jaringan.

Membuat Website Sederhana

Membuat website sederhana menggunakan HTML dan CSS sebagai dasar pengembangan web.
